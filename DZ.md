# Подсказка по Git

Создание репозитория:
```sh
git init
```
Добовавленине файла в Git
```sh
git add
```
Создание коммита:
```sh
git commit -m "Messeng"
```
Просмотр коммитов:
```sh
git log
```
Просмотр коммитов в сокращённом варианте:
```sh
git log --oneline
```
Переход к коммиту:
```sh
git checkout
```
